# Learning Python for AI

A public learning archive for building technical foundations in Python, data handling, local AI workflows and lightweight prototyping.

This repository documents my progress from basic coding concepts toward practical AI-supported tools and experiments.

## Why this exists

I work at the intersection of AI adoption, UX, internal communication and governance.

To make AI systems more operational, I am building a stronger technical foundation step by step: Python basics, data structures, files, prompts, APIs, local workflows and eventually small AI prototypes.

This repository is intentionally a learning space.

## Learning goals

- understand Python fundamentals
- work with text files, CSV, JSON and Markdown
- clean and structure data
- build small helper scripts
- understand how local AI workflows are structured
- prepare data for RAG experiments
- document technical learning clearly
